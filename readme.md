# Ecommerce_app
## setup
1. Clone the repo: `https://github.com/ammar-amk/Ecommerce_app.git`

2. from the same (clone) directory, run from terminal:

    ```sh
    npm i
    ```

5. To setup database run:
    ```sh
    npm run db:create
    ```
    
6. To start the app, run:
    ```sh
    npm run dev
    ```

## Building

```sh
npm run build
```
